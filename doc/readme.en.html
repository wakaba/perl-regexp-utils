Regular expression related modules: Regexp::Parser::Perl58, Regexp::Parser::JavaScript, and Regexp::Visualize::Simple

What's new?

See commit feed.

Modules

  • Regexp::Parser::Perl58: A Perl 5.8 regular expression parser.
  • Regexp::Parser::JavaScript: A JavaScript regular expression parser.
  • Regexp::Visualize::Simple: A regular expression visualizer.

Applications

  • Regular expression visualizer: A Web interface for Regexp::Visualize::Simple. Its source code is also available.
  • WebHACC: A Web document conformance checker, whose supported media types include JavaScript regular expressions.

Dependency

  • Perl 5.10.0 or later
  • Regexp::Parser: For Regexp::Parser::* modules.
  • Graph::Easy: For Regexp::Visualize::Simple.
